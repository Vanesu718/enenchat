## 🔧 修复内容

### 朋友圈封面图加载问题
**问题：** 朋友圈封面图上传后保存了，但刷新页面后消失

**原因：** `loadMomentsCover()` 函数在 `window.onload` 中的调用位置不对，导致DOM元素还未加载完成

**解决方案：**
```javascript
// 将 loadMomentsCover() 提前到其他资源加载之前
window.onload = async () => {
  // ... IndexedDB 初始化 ...
  updateTime();
  loadApiConfig();
  restoreDockIconsOnLoad();
  loadSavedTheme();
  loadThemeSettings();
  loadMemorySettings();
  loadMomentsCover(); // ✅ 提前加载，确保DOM元素存在
  
  // ... 其他初始化代码 ...
}
```

## 📊 存储空间管理

### 自动清理机制
当存储空间不足时，系统会自动：
1. 清理超过50条的聊天记录（保留最近50条）
2. 清理超过30条的朋友圈（保留最新30条）
3. 清理超过15条的世界书条目（保留前15条）

### 存储空间监控
- 使用 `checkStorageUsage()` 函数监控
- 当使用超过80%时会提示用户
- 大多数浏览器限制为5MB

### 图片压缩
- 超过2MB的图片自动压缩到800px以内
- 压缩质量为0.7（JPEG格式）
- 使用Canvas API进行压缩
