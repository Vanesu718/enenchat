## 一、快速开始

### 1.1 如何打开应用

- **电脑端：** 直接双击 `index.html` 文件即可在浏览器中打开，无需服务器。
- **手机端：** 
  1. 将项目部署到 GitHub Pages 或任意静态托管服务
  2. 用手机浏览器打开链接
  3. 可选：在 Safari / Chrome 中选择「添加到主屏幕」，像原生 App 一样使用

### 1.2 首次使用流程

1. 打开应用后，默认进入「我」页面
2. 前往 **通讯录** → **聊天 API 设置**，填写你的 API Key 和 Base URL
3. 回到 **对话** 页面，点击右上角 **`+`** 创建你的第一个角色
4. 点击角色进入聊天，开始与 AI 对话！

## 二、界面总览

应用底部有 **四个导航标签**，点击可切换页面：

| 图标位置 | 页面名称 | 主要功能 |
|---------|---------|---------|
| 第1个 | **对话** | 联系人列表、聊天入口、分组管理、创建角色/群聊 |
| 第2个 | **通讯录** | API设置、文风设置、世界书、用户面具、记忆设置、美化设置、个人设置 |
| 第3个 | **发现** | 朋友圈、论坛、表情管理 |
| 第4个 | **我** | 个人主页（头像、昵称、签名、照片墙、背景图） |

> 💡 底部导航图标可以在「美化设置 → 主页面设置」中自定义替换。

## 三、「我」页面

### 📍 位置：底部导航栏 → 第4个图标

这是你的个人主页，展示你的虚拟身份信息。

### 3.1 修改头像

- **操作：** 点击页面中央的圆形头像图片
- **说明：** 会弹出文件选择器，选择本地图片上传（建议 2M 以内）

### 3.2 修改昵称

- **操作：** 前往 **通讯录** → **美化设置** → **主页面设置** → 修改「用户ID (昵称)」
- **保存：** 修改后点击页面底部的「保存全部美化」

### 3.3 修改个性签名

- **操作：** 前往 **通讯录** → **美化设置** → **主页面设置** → 修改「个人签名」
- **保存：** 修改后点击页面底部的「保存全部美化」

### 3.4 照片墙

- **位置：** 「我」页面头像下方有两个照片位
- **操作：** 点击空白照片区域上传图片，下方输入框可添加备注文字

### 3.5 背景图

- **操作：** 前往 **通讯录** → **美化设置** → **主页面设置** → 「"我"页面背景图片」
- **支持：** 本地上传（2M以内）或链接上传

## 四、对话管理

### 📍 位置：底部导航栏 → 第1个图标（对话）

### 4.1 创建新角色

1. 点击对话页面右上角的 **`+`** 按钮
2. 进入「新建角色」页面，填写以下信息：

| 字段 | 说明 |
|-----|------|
| **头像** | 支持本地上传或链接上传 |
| **角色名** | 给角色起个名字 |
| **分组** | 选择所属分组（默认为"默认"，可在分组管理中新建分组） |
| **人设** | 角色的性格、外貌、习惯等详细描述。支持导入 txt/docx 文档（1M以内） |
| **核心设定与环境备忘录** | 用 1-2 句话概括核心人设及当前状态。也可点击「🤖 AI 一键提取」从人设中自动提炼 |

3. 点击「创建角色」完成

### 4.2 创建群聊

1. 点击对话页面右上角的 **「群聊」** 按钮
2. 设置群头像、群名称
3. 从已有联系人中勾选群成员
4. 点击「创建群聊」

### 4.3 分组管理

1. 点击对话页面右上角的 **「分组」** 按钮
2. 输入分组名称，点击「添加」创建新分组
3. 已有分组会显示在列表中，可进行管理

### 4.4 进入聊天

- 在对话列表中，点击任意联系人即可进入聊天页面

## 五、聊天功能（核心）

### 📍 位置：对话列表 → 点击联系人进入

### 5.1 发送消息

- 在底部输入框中输入文字
- 按 **Enter** 键 或点击右侧 **纸飞机发送按钮** 发送
- 支持 **Shift + Enter** 换行

### 5.2 AI 回复

- 点击发送按钮右侧的 **AI 回复按钮**（带有 AI 图标）
- AI 会根据角色设定、聊天上下文生成回复
- 需要提前在「通讯录 → 聊天 API 设置」中配置好 API

### 5.3 线上/线下模式切换

- **位置：** 聊天页面右上角 **「···」菜单** → 顶部的模式开关
- **线上模式：** AI 回复模拟即时通讯（适合聊天对话）
- **线下模式：** AI 回复模拟面对面交流（适合叙事/剧情）

### 5.4 附件面板

点击输入框左侧的 **📎 附件按钮**，展开附件面板：

| 功能 | 说明 |
|-----|------|
| **日记** | 发送日记格式的消息 |
| **图片** | 上传图片发送到聊天中 |
| **图片(文字)** | 上传图片并自动识别其中的文字内容 |
| **语音** | 发送语音消息 |
| **红包** | 发送红包样式的消息 |
| **转账** | 发送转账样式的消息 |
| **视频** | 视频功能（敬请期待） |
| **更多** | 更多功能（敬请期待） |

### 5.5 表情包面板

- 点击输入框左侧的 **😊 表情按钮**
- 会弹出底部表情面板，按组名分类显示
- 点击表情即可发送
- 表情包可在 **发现 → 表情管理** 中添加

### 5.6 消息操作

**长按/点击消息**可触发以下操作：

| 操作 | 说明 |
|-----|------|
| **编辑** | 修改已发送的消息内容 |
| **删除** | 删除单条消息 |
| **引用回复** | 引用某条消息进行回复，被引用内容会显示在输入框上方 |

### 5.7 批量删除消息

1. 点击聊天页面右上角 **「···」菜单** → **「批量删除」**
2. 勾选要删除的消息
3. 底部显示已选数量，点击「删除」确认

### 5.8 快速重 Roll

- **位置：** 聊天页面右上角 **「···」菜单** → **「快速重roll」**
- **说明：** 重新生成 AI 的最后一条回复

### 5.9 聊天搜索

- 点击聊天页面右上角的 **🔍 搜索图标**
- 输入关键字，搜索当前聊天的所有消息记录
- 点击搜索结果可定位到对应消息

### 5.10 状态卡片

- **位置：** 点击聊天页面顶部的 **角色名称**
- **展示内容：**
  - 📍 **地点** — 角色当前所在位置
  - 😊 **心情** — 角色当前情绪状态
  - 💭 **心声** — 角色内心独白
  - ❤️ **好感度** — 进度条 + 百分比显示，可点击「清零」重置

### 5.11 清空聊天记录

- **位置：** 聊天页面右上角 **「···」菜单** → **「清空记录」**
- ⚠️ 此操作不可逆，请谨慎操作

### 5.12 删除联系人

- **位置：** 聊天页面右上角 **「···」菜单** → **「删除联系人」**
- ⚠️ 删除后联系人及所有聊天记录将被清除

### 5.13 短期记忆查看

- **位置：** 聊天页面右上角 **「···」菜单** → **「短期记忆」**
- 查看当前聊天的短期记忆列表（显示条数如 `3/10`）
- 支持批量删除短期记忆条目

## 六、聊天设置

聊天设置分为三个标签页：

### 6.1 基础功能

| 功能 | 说明 |
|-----|------|
| **隐藏双方头像** | 开启后聊天界面不再显示头像 |
| **个性化聊天背景** | 为当前聊天设置专属壁纸，支持上传图片或恢复默认 |

### 6.2 角色设定

| 功能 | 说明 |
|-----|------|
| **自定义我的头像** | 仅在当前聊天中使用特定头像 |
| **用户面具选择** | 从已保存的用户面具中快速切换，自动替换头像、昵称和人设 |
| **我的聊天昵称** | 设置仅在当前聊天中使用的昵称（留空则使用全局昵称） |

| **联系人分组** | 修改当前联系人所属分组 |
| **联系人头像和名字** | 修改联系人的头像和显示名称 |

### 6.3 记忆与高阶

| 功能 | 说明 |
|-----|------|
| **关联世界书** | 勾选要在当前聊天中使用的世界书条目（支持多选） |

## 七、通讯录 / 设置页面

### 📍 位置：底部导航栏 → 第2个图标（通讯录）

### 7.1 聊天 API 设置

**路径：** 通讯录 → 聊天API设置

这是使用 AI 功能的前提配置。

| 字段 | 说明 |
|-----|------|
| **API Key** | 你的 API 密钥（如 `sk-...`） |
| **Base URL** | API 的基础地址（默认 `https://api.openai.com/v1`） |
| **模型** | 选择 AI 模型。点击「拉取」按钮可从 API 自动获取可用模型列表 |
| **温度** | 0~1 之间，越低回复越稳定，越高越有创意 |
| **流式输出** | 开启后 AI 回复会有打字机逐字输出效果 |

**预设管理：**
- **另存为预设：** 将当前配置保存为命名预设
- **加载预设：** 从下拉框选择已保存的预设
- **删除预设：** 删除当前选中的预设
- **保存当前配置：** 保存为默认使用的配置

### 7.2 文风设置

**路径：** 通讯录 → 文风设置

管理 AI 回复的文字风格。

**创建文风：**
1. 点击右上角 **`+`** 进入「新增文风设定」
2. 填写文风名称
3. 在「文风设定内容」中输入详细描述（字数不限），或点击「📁 导入文档」从 txt/docx 文件导入
4. 点击「✨ 一键提取」— AI 会消耗一次 API 调用自动压缩提炼文风内容
5. 在「AI 提炼预览」中查看/编辑压缩后的内容
6. 点击「保存」

> ⚠️ 保存文风时会消耗一次 API 进行智能压缩，以节省后续聊天的 Token 消耗。

### 7.3 世界书管理

**路径：** 通讯录 → 世界书管理

世界书用于存储世界观设定、记忆总结等长期信息。

**分类标签：**
- **全部** — 显示所有条目
- **记忆总结** — AI 的长期记忆归档
- **世界观** — 背景设定、世界规则
- **其他** — 自定义内容

**创建条目：**
1. 点击右上角 **`+`** 进入「新增世界书条目」
2. 填写名称和分类
3. 选择触发方式：
   - **常驻（全局生效）：** 每次对话都会发送给 AI
   - **关键词触发：** 仅当聊天记录中包含指定关键词时才发送（推荐，节省 Token）
4. 输入世界观设定内容，或导入 txt/docx 文档
5. 点击「保存条目」

### 7.4 用户面具设置

**路径：** 通讯录 → 用户面具设置

用户面具让你可以快速切换不同身份。

**创建面具：**
1. 点击右上角「新建」
2. 上传面具头像（本地或链接）
3. 输入面具 ID（名称）
4. 输入人物设定
5. 点击「保存面具」

**使用面具：** 在聊天设置的「用户面具选择」中快速应用。

### 7.5 记忆设置

**路径：** 通讯录 → 记忆设置

管理 AI 的长期记忆和短期记忆系统。

#### 长期记忆 (LTM)

| 设置项 | 说明 |
|-------|------|
| **开启自动总结** | 开关控制是否自动归档记忆 |
| **触发机制** | 当短期记忆累计到 10 条时，自动触发长期记忆归档 |
| **总结提示词** | 可自定义 AI 生成总结时使用的提示词，支持变量替换 |

**可用变量：**
- `{charName}` → 替换为当前角色名称
- `{userName}` → 替换为当前用户昵称
- `{timeHeader}` → 替换为系统时间段标记（如 `【25/01/01 07:00 - 11:00】`）

> 🔄 容错机制：如果某次总结失败，系统不会清空历史对话，而是保留下来，在下一轮对话时自动重试。

#### 短期记忆 (STM)

| 设置项 | 说明 |
|-------|------|
| **开启短期记忆** | 开关控制 |
| **短期记忆容量** | 每多少轮对话生成一条短期记忆（1-30轮，默认10轮） |
| **短期记忆提示词** | 用于生成摘要的提示词 |

修改完成后点击 **「💾 保存记忆设置」**。

### 7.6 美化设置

**路径：** 通讯录 → 美化设置

美化设置分为四个子菜单：

#### 7.6.1 主页面设置

| 功能 | 说明 |
|-----|------|
| **用户ID (昵称)** | 修改你的昵称 |
| **个人签名** | 修改个性签名 |
| **底部图标替换** | 替换四个底部导航图标（仅支持1M以内图片） |
| **美化备份与恢复** | 单独导出/导入美化相关设置 |
| **"我"页面背景图片** | 上传或链接设置背景图（2M以内） |

修改后点击 **「保存全部美化」**。

#### 7.6.2 主题调色盘

| 功能 | 说明 |
|-----|------|
| **选择主题** | 三种预设主题：默认粉色 / 线下简约 / 暗黑模式 |
| **主题调色盘** | 6种预设色（粉、蓝、绿、紫、黄、红）一键切换 |
| **自定义颜色** | 使用颜色选择器或输入 HEX 色值 |

#### 7.6.3 气泡调色盘

| 功能 | 说明 |
|-----|------|
| **对方气泡颜色** | 设置左侧（对方）消息气泡的颜色 |
| **我的气泡颜色** | 设置右侧（我的）消息气泡的颜色 |
| **气泡角度** | 调整气泡圆角大小（0-30px） |
| **透明度** | 调整气泡透明度（0.1-1.0） |
| **对方/我的气泡装饰** | 上传 PNG 装饰图，可调整位置（四角选择）、大小、水平/垂直偏移 |

#### 7.6.4 叙事美化

针对聊天消息中不同类型的文字设置样式：

| 文字类型 | 说明 | 示例 |
|---------|------|------|
| **普通正文** | 叙事描写文字 | `比格林看着用户` |
| **"" 包裹文字** | 对话文字 | `"怎么这么多代码要修！"` |
| **{} 包裹文字** | 动作/心理描写 | `{它叼起扳手跑走。}` |

每种类型可分别设置：
- 文字颜色 / 文字底色
- 加粗 (B) / 斜体 (I)

其他叙事美化功能：
- **自定义全局字体** — 输入字体文件链接（.ttf, .woff），点击「应用」
- **全局字体大小** — 拖动滑块调节（12px - 24px）

修改后点击 **「保存叙事美化设置」**。

### 7.7 个人设置

**路径：** 通讯录 → 个人设置

| 功能 | 说明 |
|-----|------|
| **💬 仅聊天备份** | 导出联系人、聊天记录（不含图片）及世界书记忆 |
| **📦 全局备份** | 导出全部数据（含 API、世界书、主题、图片等） |
| **📥 导入备份** | 从 JSON 文件恢复数据（聊天版/全局版均可） |
| **⚠️ 恢复出厂设置** | 清空所有数据，**此操作不可逆！** |

## 八、发现页面

### 📍 位置：底部导航栏 → 第3个图标（发现）

### 8.1 朋友圈

**路径：** 发现 → 朋友圈

#### 浏览朋友圈
- 进入后看到顶部封面图（可点击更换）
- 点击 **🔄 刷新按钮** 加载/刷新朋友圈内容
- AI 角色会根据设定自动生成朋友圈动态

#### 发布朋友圈
1. 点击朋友圈页面右上角的 **`+`** 按钮
2. 输入朋友圈内容
3. 可选择「发布面具」— 选择以哪个身份发布
4. 设置「谁可以看」：
   - **公开** — 所有朋友可见
   - **私密** — 仅自己可见
   - **部分可见** — 选择特定朋友/分组可见
   - **不给谁看** — 选择不让特定朋友/分组看到
5. 可上传图片：本地上传限1张（2M以内），链接上传最多4张
6. 点击「发表」

#### 更换朋友圈封面
- 直接点击朋友圈顶部的封面区域即可选择新图片

#### 朋友圈设置
- 点击页面右上角「设置」查看可见性说明

### 8.2 论坛

**路径：** 发现 → 论坛

论坛分为多个板块，默认包含三个主题板块：

| 板块 | 主题色 | 说明 |
|-----|-------|------|

## 十一、常见问题 FAQ

### Q1：AI 没有回复 / 报错？
**A：** 请检查以下项：
1. 是否已在「通讯录 → 聊天 API 设置」中填写了正确的 API Key 和 Base URL
2. API Key 是否有效（余额是否充足）
3. 选择的模型是否可用
4. 打开浏览器控制台（F12）查看具体错误信息

### Q2：图片上传失败？
**A：** 
- 本地上传的图片大小限制：文档 1M，图片 2M
- 大量图片可能导致 localStorage 存储空间不足（通常限制 5-10MB）
- 建议使用链接上传或压缩图片

### Q3：数据会丢失吗？
**A：** 
- 数据存储在浏览器的 localStorage 中
- 清除浏览器数据会导致所有数据丢失
- **强烈建议定期使用「个人设置 → 全局备份」导出数据**

### Q4：如何在手机上使用？
**A：** 
1. 将项目部署到 GitHub Pages 等静态托管平台
2. 用手机浏览器打开
3. Safari：点击分享 → 添加到主屏幕
4. Chrome：菜单 → 添加到主屏幕

### Q5：支持哪些文档格式导入？
**A：** 
- ✅ `.txt` 纯文本文件
- ✅ `.docx` Word 2007+ 文档
- ❌ `.doc` 旧版 Word（不支持）
- ❌ `.wps` 文件（不支持）

### Q6：文风设置为什么要消耗一次 API？
**A：** 系统会使用 AI 对你输入的详细文风描述进行智能压缩提炼，生成精简版本。这样在后续每次聊天中都能节省大量 Token 消耗，是一次性投入、长期收益的设计。

### Q7：世界书的「常驻」和「关键词触发」有什么区别？
**A：** 
- **常驻：** 每次对话都会将该世界书条目发送给 AI，适合核心世界观设定
- **关键词触发：** 只有当最近聊天中出现指定关键词时才发送，可以节省 Token 并减少 AI 产生幻觉
- **推荐：** 对于非核心设定，建议使用关键词触发

### Q8：如何让 AI 更好地记住之前的对话？
**A：** 
1. 开启「记忆设置」中的短期记忆和长期记忆
2. 为角色编写「核心设定与环境备忘录」
3. 在世界书中添加重要的记忆总结
4. 将相关世界书条目关联到聊天中
